##  Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/yadavshubham01/tuf_assign.git
cd financial Dashboard
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Development Server

```bash
npm run dev
```

App will be available at:
 `http://localhost:5173`
